  <h1>Vintage</h1>
  The video library of vintage content.
</div>

## **How to install and run locally ?**

```
$ https://github.com/rdshinde/vintage-video-library.git
$ npm install
$ npm start
```
## **Features -**

- Home page
- Video listing page
- Filter Videos
- Individual video page
- Like video 
- Watch Later Video
- History video
- Create playlist
- Authentication:
  -  Signup
  -  Login

## **Built Using -**

- React JS
- React Context API + useReducer
- React Router v6
- Vanilla CSS and Bermuda CSS.
- MockBee
